# Mos

![Xcode 9.0+](https://img.shields.io/badge/Xcode-9.0%2B-blue.svg)
![Swift 4.0+](https://img.shields.io/badge/Swift-4.0%2B-orange.svg)

A simple tool to smooth and reverse your mouse scroll on MacOS, handy and free.

[中文](https://github.com/Caldis/Mos/blob/master/README.md) | [English](https://github.com/Caldis/Mos/blob/master/README.enUS.md)


## Homepage

http://mos.caldis.me/


## Features

- Smooth the mouse wheel scroll just like touchpad
- Distinguish the touchpad scroll and mouse wheel scroll, reverse the scroll position separatily.
- A mointor can render the scroll event.
- Custom the scroll effect, the speed and easing effect.
- Ignore list, black/white list mode.
- Base on swift4.

## Download & Install

### Homebrew

If you wish to install the application from [Homebrew](https://brew.sh):

```bash
$ brew cask install mos
```

The application will live at `/Applications/Mos.app`.

To update the app:

```bash
$ brew update
$ brew cask reinstall mos
```

Quit the app then relaunch the application.
