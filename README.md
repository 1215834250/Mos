# Mos

一个用于在MacOS上平滑你的鼠标滚动效果的小工具, 让你的滚轮丝滑如触控板。


## 特性

- 支持分离触控板/鼠标事件, 单独翻转鼠标滚动方向。
- 一个用于监控滚动事件的图形化呈现窗口。
- 基于Swift3构建。
